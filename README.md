# Temperature Converter

## Objective

To develop a web application that converts temperature values between Celsius, Fahrenheit, and Kelvin accurately.

## Steps Performed

1. Created the webpage structure using HTML.
2. Designed a responsive user interface using CSS.
3. Implemented temperature conversion logic using JavaScript.
4. Added input validation and result display.
5. Tested the application with different temperature values.

## Tools Used

* HTML
* CSS
* JavaScript
* VS Code
* GitHub

## Outcome

Successfully developed a temperature converter that performs quick and accurate temperature conversions between different units.
